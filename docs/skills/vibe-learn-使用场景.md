# `/vibe-learn` 使用场景

## 一句话

`/vibe-learn` 是一个为 vibe coder/非科班开发者设计的技能教学工具，只教"怎么用"和"怎么向 AI 描述需求"，不教底层原理和数学推导。

## 什么时候用它

1. **学新工具/库/框架的上手用法** — 比如"怎么用 Prisma 连数据库"、"怎么用 Zustand 做状态管理"，不求懂源码，只求跑起来
2. **不知道怎么向 AI 描述需求** — 知道想要什么效果，但不知道用哪些术语、怎么写 Prompt 才能让 AI 一次性生成对的东西
3. **遇到开发技能盲区** — 比如想加个 CI/CD、想写测试、想部署项目，之前没做过，需要一套可直接复制执行的步骤模板

## 什么时候不用它

- 纯理论问题（"这个算法的时空复杂度"、"MVVM 和 MVC 的本质区别"）
- 底层原理探究（"V8 引擎怎么优化闭包"）
- 数学推导（"Transformer 的注意力机制证明"）
